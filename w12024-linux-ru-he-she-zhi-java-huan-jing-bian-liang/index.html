【W01/2024】Linux 如何设置 Java 环境变量

2024-01-01

  1. 打开终端，以管理员或root权限登录。

  2. 确认您已经安装了Java开发工具包（JDK）。您可以使用以下命令检查Java是否已安装：

    java -version

    如果Java未安装，可以使用包管理器（如apt、yum等）安装Java。

  3. 设置JAVA_HOME变量：

    • 确定JDK的安装路径。您可以使用以下命令找到JDK的安装路径：

      readlink -f $(which java) | sed "s:bin/java::"

    • 打开终端配置文件（如.bashrc、.bash_profile、.zshrc等），可以使用以下命令打开：

      vi ~/.bashrc

    • 在文件末尾添加以下行，将JDK的安装路径替换为您在第一步中找到的路径：

      export JAVA_HOME=/path/to/your/jdk
export PATH=$JAVA_HOME/bin:$PATH

    • 保存并关闭文件。

  4. 更新环境变量：

    • 执行以下命令，使新的环境变量生效：

      source ~/.bashrc

  5. 验证设置是否成功：

    • 使用以下命令检查JAVA_HOME是否正确设置：

      echo $JAVA_HOME

    • 使用以下命令检查Java版本是否正确：

      java -version

现在，您已经成功设置了Java变量。您可以在终端中运行Java命令，并在开发环境中使用JAVA_HOME变量。
